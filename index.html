About the course

An applied 8‑week course covering LLM fundamentals, text and data workflows, literature synthesis, custom chatbots and retrieval-augmented systems, plus automation for documents and spreadsheets. Students build tools and submit weekly deliverables culminating in a capstone project.

Sample weekly topics

  1. LLM fundamentals & prompt design
  2. Generating and editing documents & presentations
  3. Automating spreadsheets and data tasks
  4. Literature search & knowledge synthesis
  5. Text analysis & classification
  6. Building RAG systems & chatbots
  7. Workflow automation & integrations
  8. Capstone: deployable project

Course outcomes

  • Design reproducible AI-assisted workflows for research and reports
  • Build simple RAG-backed assistants for domain tasks
  • Automate repetitive tasks across docs, spreadsheets and emails
  • Evaluate and choose appropriate GenAI tools for a given task
Capstone

Students deliver a small, deployable project demonstrating a workflow (e.g., automated report generator, literature assistant, or chatbot for a research task).
